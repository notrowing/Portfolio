# My Portfolio

### 주요 기능
1. ABOUT ME : 자기소개
2. SKILL : 보유기술
3. PROJECT : 팀 프로젝트 나열
4. CHANNEL : 티스토리, 깃헙 
5. CONTACT : 문의하기(EmailJS 사용) 및 이메일주소 안내
